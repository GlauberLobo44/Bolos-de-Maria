# Bolos-de-Maria
Um site de gerenciamento de pedidos para uma confeiteira que a permita organizar os pedidos recebidos.

#Funcionalidades
O site conta com as funcões de Criar e editar pedidos por meio de um calendário interativo, além da funcionalidade de gerenciar o cardápio e a lista de clientes
O aplicativo salva dos dados no armazenamento local do navegador
